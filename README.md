# cat-and-notcats
Neural network that identifies whether an image is a cat or not

## Libraries
- TensorFlow (GPU)
- Keras
- PIL
- numpy

## How to Use
For arg_test.py:
```
python arg_test.py -m [MODEL] -i [IMAGE]
```

For rand_test.py
```
python rand_test.py -m [MODEL] -d [DIRECTORY]
```

Don't run main.py unless you're training a new network all together.
